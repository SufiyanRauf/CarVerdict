# CarVerdict

A retrieval-augmented (RAG) question-and-answer app over public NHTSA vehicle-safety data: owner complaints, recalls, and safety ratings. Ask plain-English questions about a vehicle, compare cars, and see how defects trend across model years, with answers grounded in real complaint records.

## Tech stack
- Next.js (App Router) and React
- Google Gemini (chat and embeddings)
- Pinecone (vector search)
- NHTSA public APIs (complaints, recalls, safety ratings, VIN decode)
